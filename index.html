I14Y Toolbox

The I14Y Toolbox contains experimental helper applications for the I14Y Interoperability Platform, Switzerland's governmental data catalog. These tools facilitate efficient data imports and exports.

I14Y Excel Upload

Easily import your Excel-based dataset inventory to the I14Y Interoperability Platform with this tool.

I14Y Concept Import

Easily import concepts from a spreadsheet table to the I14Y Interoperability Platform with this tool.

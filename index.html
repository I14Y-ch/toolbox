I14Y Toolbox

The I14Y Toolbox contains experimental helper applications for the I14Y Interoperability Platform, Switzerland's governmental data catalog. These tools facilitate efficient data imports and exports.

All webapps are experimental, quickly built, and often altered. There is no warranty provided. Depending on the application, hosting may currently not be on the infrastructure of the confederation. In some cases, it might be better to run the application on premise. The codes and instructions are available in the GitHub repository of I14Y.

I14Y Excel Upload

Easily import your Excel-based dataset inventory to the I14Y Interoperability Platform with this tool.

I14Y Keyword Generator

Create keywords to enhance your dataset descriptions on I14Y.

I14Y Concept Import

Easily import concepts from a spreadsheet table to the I14Y Interoperability Platform with this tool.

Swagger2I14Y

Use your technical API specification (Swagger) to generate an API description on I14Y.

SHACL-Generator

Generate the description of a data structure in the SHACL format. Use your actual data to start with. The generated File can be uploaded to I14Y.
